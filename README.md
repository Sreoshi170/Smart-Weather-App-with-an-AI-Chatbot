

# 🌦️ Weather App with AI Chatbot

An interactive **Weather Application with Chatbot Integration** that provides real-time weather updates along with a smart conversational interface. This project combines **frontend design + API integration + AI-like interaction** to enhance user experience.

---

## 🚀 Features

✨ **Real-Time Weather Data**

* Fetches live weather information using an API
* Displays temperature, weather condition, and description

🤖 **Chatbot Integration**

* Interactive chatbot interface for user queries
* Enhances usability and engagement

🌍 **Location-Based Search**

* Users can search weather by city name
* Dynamic updates based on input

🎨 **Modern UI Design**

* Built using **Tailwind CSS**
* Clean, responsive, and visually appealing interface

📊 **Detailed Weather Information**

* Temperature
* Weather condition
* Additional metrics (like humidity, etc.)

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Styling:** Tailwind CSS
* **API:** Weather API (stored in `api.txt`)
* **Design:** Responsive UI with modern layout

---

## ⚙️ How to Run the Project

1. **Clone the repository**

```bash
git clone https://github.com/your-username/weather-app.git
```

2. **Navigate to project folder**

```bash
cd weather-app
```

3. **Open the project**

* Simply open `index.html` in your browser

---

## 🔑 API Setup

* Add your **Weather API Key**
* Replace it in the JavaScript code inside `index.html`
